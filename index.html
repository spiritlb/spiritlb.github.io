GitHub+Hexo个人博客的搭建
今天发现一个很好玩的博客！
GitHub+Hexo个人博客的搭建，因为个人技术菜，搞了几天才算完成，现在先简单说一下这个GitHub吧。
GitHub 是一个面向开源及私有软件项目的托管平台，因为只支持 Git 作为唯一的版本库格式进行托管，故名 GitHub。
GitHub除了Git代码仓库托管及基本的Web管理界面以外，还提供订阅、讨论组、文本渲染、在线文件编辑器、协作图谱（报表）、代码片段分享（Gist）等功能。
了解到的也就这么多。下面我们来说说如何实现的吧。
1.安装

2.安装和配置好Git环境，安装成功后就是在电脑任意文件夹鼠标右键出现

3.创建GitHub账号（最好用英文）和新建项目，项目命名格式必须为：账户名.github.io
    创建新项目

    项目命名格式：    账户名.github.io
                    勾选Initialize this repository with a README

    4.安装Hexo，我在E盘创建了一个blog文件夹，运行cmd，到该目录下
        4.1跳转到E盘直接输入e:+回车；cd blog+回车
        4.2输入npm install hexo -g，开始安装Hexo
        4.3输入hexo -v，检查hexo是否安装成功
        4.4输入hexo init，初始化该文件夹
        看到出现Start blogging with Hexo！表示已经安装成功
        4.5输入npm install，安装所需要的组件
        4.6输入hexo g，首次体验Hexo
        4.7输入hexo s，开启服务器，访问该网址，正式体验Hexo
    5. 将Hexo与Github page联系起来，设置Git的user name和email
        5.1任意文件夹鼠标右键点击Git Base Here
        5.2进入文件夹blog
        5.3输入ssh-keygen -t rsa -C “你的注册邮箱”，连续三个回车，生成密钥，最后得到了两个文件：id_rsa和id_rsa.pub    （默认存储路径是：C:\Users\Administrator\.ssh）
        5.4输入eval "$(ssh-agent -s)"，添加密钥到ssh-agent
        5.5输入cd ~/.ssh，检查是否由.ssh的文件夹
        5.6输入ls，列出该文件下的内容。
        5.7再输入ssh-add ~/.ssh/id_rsa，添加生成的SSH key到ssh-agent
        5.8登录Github，点击头像下的settings，添加ssh
        5.9新建一个new ssh key，将id_rsa.pub文件里的内容复制上去
        5.10输入ssh -T git@github.com，测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了
        6配置Deployment，在其文件夹中，找到_config.yml文件，修改repo值
            deploy:
                type:git
                repository:你创建的库的URL：我的是git@github.com: spiritlb/spiritlb.github.io.git
                branch: master
        7.1 安装扩展npm install hexo-deployer-git --save
        7.2新建博客hexo new post “博客名”
        7.3部署hexo d –g
        7.4部署成功后访问你的地址：http://用户名.github.io
